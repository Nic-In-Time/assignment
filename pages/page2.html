What is a computer?

Computers are devices that process information according to instructions (code) and are multipurpose.

Hardware

Hardware is the physical part of the computer, so the physical components. It can wear out over time. Some examples of hardware are:

  1. Keyboard
  2. Mouse
  3. Monitor
  4. Graphical Processing Unit (GPU)
  5. Computing Processing Unit (CPU)
  6. Motherboard
  7. Random Access Memory (RAM)
  8. Storage
  9. Power Supply

Software

Software is a set of instructions written using a variety of programming languages to help control hardware. It does not wear out over time, but it can be affected by bugs or glitches. It is intangible. Some examples of software, among millions of others, are:

  1. Linux
  2. MacOS
  3. Windows
  4. VS Code
  5. Godot Game Engine
  6. Word
  7. Excel

Terminals, Folder Structures and File Systems

Folder Structures and File Systems

Folders follow an inverted tree-like structure. Most folders contain other folders.

Terminals

Terminals allow users of a computer to input commands. These commands can do many different things, such as:

  1. ls
    Lists all folders and files in the Current Working Directory (CWD)
  2. mkdir
    Creates a directory
  3. rmdir
    Deletes a directory
  4. cd
    Change CWD
  5. TAB
    Autocompletes file or folder name while typing a command
  6. cd ..
    Goes up a level in the file system
  7. mv
    Deletes a directory
  8. less
    Prints file to command line, press "q" to quit
    9. >
  9. nano
    Edit a file in nano. ctrl + O to save, ctrl + X to exit
